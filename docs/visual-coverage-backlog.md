## Visual Standard

Each target page should have at least one page-specific visual that teaches the
main idea without requiring the reader to finish the paragraph first. Technical
topics should favor diagrams, flows, ledgers, and system maps. Reflective topics
should favor memorable metaphor images. Reference artifacts should get both a
recall image and a mechanism diagram when the content supports it.
